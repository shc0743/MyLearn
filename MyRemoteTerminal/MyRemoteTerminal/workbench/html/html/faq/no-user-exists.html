No user account exists

"No user account exists" means there is not any user account in remote config.
This means that you didn't config the service after set up, or the config file is corrupted.
If you are system administrator and you didn't config the service, you should access config page and follow the prompt.
If you are sure about the configuration, you should check the config file.
If the configuration file is corrupted, you should Delete it and create a new one.
